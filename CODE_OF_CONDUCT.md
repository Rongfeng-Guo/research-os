# Code of Conduct

## Our standard

Participation in this project should be respectful, professional, and constructive.

Examples of expected behavior:

- giving actionable technical feedback
- assuming good intent while reviewing contributions
- focusing criticism on the code or design, not the person
- respecting different levels of experience and backgrounds

Examples of unacceptable behavior:

- harassment, discrimination, or personal attacks
- intentionally disruptive conduct
- publishing private information without permission
- bad-faith technical discussion

## Scope

This code of conduct applies to project spaces, issue discussions, pull requests, and other community interactions tied to this repository.

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, issues, or contributions that violate this code of conduct.

## Reporting

Report conduct issues to the project maintainer through a private channel rather than a public issue.
